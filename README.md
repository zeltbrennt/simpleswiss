# Readme

Simple Swiss Chess Tournament Creator

- Add Names to the Tournament (must be even number).
- Start the first Round
- Save the results by clicking on the winner or the middle to indicate a draw
- Click next round to find a new pairing
  - finds the best player
  - looks for the next best player, that has not yet played against them
  - tries to keep the distribution of colors equal in the long run (on might play the same color in a row)
- Shows results of all games
- Shows table of current positions in Tournament
  - for tie-breakers the Bucholz-Score is used (the sum of points each opponent has)
